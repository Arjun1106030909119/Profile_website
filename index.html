Disaster Rescue

Web Application

In my project, I've developed a comprehensive web application tailored for Disaster Rescue operations, featuring four distinct functionalities accessible via intuitive buttons. The Incident Reporting feature empowers users to swiftly request assistance by reporting vital information about the disaster, including its type, location, and even uploading images for enhanced situational awareness. For seamless coordination, the Incident Viewing module enables users to visualize ongoing disasters, facilitating physical aid deployment and financial support through QR code donations.

Additionally, the application streamlines volunteer management by offering functionalities for both adding volunteers, where individuals can input their details such as name, mobile number, and address, and viewing volunteer profiles to match resources with needs efficiently. With these integrated features, the Disaster Rescue web application serves as a vital tool for optimizing disaster response efforts and fostering community resilience.
